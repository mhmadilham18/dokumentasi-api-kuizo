Authentication Service

Endpoint-endpoint ini menangani semua hal yang berkaitan dengan registrasi dan login untuk semua jenis pengguna (Admin, Guru, Siswa).

User & Admin Authentication

POST /register-admin

Registrasi Admin

Mendaftarkan akun admin utama. Endpoint ini idealnya hanya dapat digunakan satu kali saat sistem pertama kali di-setup.

Authorization: None

Request Body application/json

Key Tipe Deskripsi
username string Nama pengguna unik. Wajib, min 3.
full_name string Nama lengkap. Wajib, min 3.
email string Alamat email yang valid. Wajib.
nomer_induk string Nomor Induk Pegawai (NIP). Wajib, min 5.
password string Password untuk akun. Wajib, min 6.

Success Response 201 Created

{
    "status": "success",
    "message": "Admin berhasil terdaftar",
    "data": {
        "id": "b20542c9-4386-4636-b906-863f36cbc7dd",
        "email": "admin@sekolah.sch.id"
    }
}
POST /register-user

Registrasi User (oleh Admin)

Admin mendaftarkan akun baru untuk guru atau siswa.

Authorization: Bearer Token (Admin)

Request Body application/json

Key Tipe Deskripsi
username string Nama pengguna unik. Wajib, min 3.
full_name string Nama lengkap. Wajib, min 3.
email string Alamat email yang valid. Wajib.
nomer_induk string Nomor Induk Pegawai/Siswa (NIP/NIS). Wajib, min 5.
password string Password untuk akun. Wajib, min 6.
role string Peran pengguna. Pilihan: "guru" atau "siswa". Wajib.

Success Response 201 Created

{
    "status": "success",
    "message": "Akun guru berhasil terdaftar",
    "data": {
        "id": "a1b2c3d4-e5f6-1234-abcd-567890abcdef",
        "email": "guru.baru@sekolah.sch.id",
        "role": "guru"
    }
}
POST /login

Login Pengguna

Mengautentikasi semua jenis pengguna (admin, guru, siswa) dan mengembalikan access token.

Authorization: None

Request Body application/json

Key Tipe Deskripsi
email string Alamat email terdaftar. Wajib.
password string Password pengguna. Wajib.

Success Response 200 OK

{
    "status": "success",
    "message": "Login berhasil",
    "data": {
        "accessToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJpZCI6IjEyMzQ1Iiwicm9sZSI6Imd1cnUifQ.abcdef",
        "role": "guru"
    }
}
